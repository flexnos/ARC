# 🚀 START HERE - Easiest Way to Run

## ⚡ Super Simple - Just 1 Click!

### Double-click this file:
```
START_HERE.bat
```

That's it! Two windows will open:
1. **Backend Server** (port 8000)
2. **Frontend UI** (port 3000)

Then open your browser to **http://localhost:3000**

---

## 🎯 What Happens Next

### Window 1 (Backend):
- Shows Python version
- Checks dependencies  
- Starts FastAPI server
- Displays: `Uvicorn running on http://127.0.0.1:8000`

### Window 2 (Frontend):
- Shows Node.js version
- Installs packages (first time only, takes 2-3 min)
- Starts React app
- Displays: `ready - started server on 0.0.0.0:3000`

### Your Browser:
Opens to beautiful UI with:
- Animated homepage
- Feature showcase
- Upload interface
- Analytics dashboard

---

## ❗ Important Notes

1. **Keep both windows open** - Close them only when done
2. **First run takes longer** - npm installs ~350 packages
3. **Wait for "ready" message** - Don't close frontend window
4. **Use modern browser** - Chrome, Edge, or Firefox recommended

---

## 🔧 If Something Goes Wrong

Check **TROUBLESHOOTING.md** for detailed fixes!

Quick fixes:
- **Python error?** → Install Python from python.org
- **Node error?** → Install Node.js from nodejs.org  
- **Port in use?** → Close other apps or restart computer
- **Blank page?** → Wait 15 seconds, refresh browser

---

## ✅ Success Checklist

You're good when you see:

- [x] Backend window shows "Application startup complete"
- [x] Frontend window shows "ready - started server"
- [x] Browser shows animated UI at http://localhost:3000
- [x] Can navigate between Home/Evaluate/Results tabs

## 📞 Need Help?

1. Read TROUBLESHOOTING.md
2. Check error messages in the two windows
3. Verify Python and Node.js are installed
4. Try running as Administrator

**Most common fix**: Close everything and run `START_HERE.bat` again!

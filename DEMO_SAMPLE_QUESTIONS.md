## 📚 Available Subjects

### 1. 🌿 General Science
**Topic:** Photosynthesis

**Question:**
```
Explain the process of photosynthesis in plants.
```

**Sample Student Answer:**
```
Photosynthesis is how plants make their own food using sunlight, water, and carbon dioxide. They convert light energy into chemical energy stored in glucose.
```

**AI Reference Answer:**
```
Photosynthesis is the biochemical process by which green plants convert light energy into chemical energy. Using chlorophyll, they absorb sunlight and combine carbon dioxide from the air with water to produce glucose and oxygen. This process occurs primarily in the leaves within specialized structures called chloroplasts.
```

### 2. 📜 History
**Topic:** World War I

**Question:**
```
What were the main causes of World War I?
```

**Sample Student Answer:**
```
WWI was caused by militarism, alliances, imperialism, and nationalism. The assassination of Archduke Franz Ferdinand triggered the war.
```

**AI Reference Answer:**
```
World War I was caused by four main factors: Militarism (arms race between nations), Alliances (complex treaty systems), Imperialism (competition for colonies), and Nationalism (pride in one's nation). The immediate trigger was the assassination of Archduke Franz Ferdinand of Austria-Hungary in Sarajevo on June 28, 1914, by Gavrilo Princip, a Serbian nationalist.
```

### 3. ⚡ Physics
**Topic:** Newton's Laws

**Question:**
```
State Newton's Second Law of Motion and provide an example.
```

**Sample Student Answer:**
```
F=ma. Force equals mass times acceleration. Example: pushing a car requires more force than pushing a bicycle.
```

**AI Reference Answer:**
```
Newton's Second Law states that the force acting on an object is equal to the mass of that object multiplied by its acceleration (F=ma). This means the acceleration of an object depends on two variables: the net force acting upon it and its mass. For example, applying 10N of force to a 2kg ball will accelerate it at 5 m/s², while the same force applied to a 10kg box will only accelerate it at 1 m/s².
```

### 4. 💻 Computer Science
**Topic:** Memory Types

**Question:**
```
Explain the difference between RAM and ROM in computers.
```

**Sample Student Answer:**
```
RAM is volatile memory that stores temporary data. ROM is non-volatile and stores permanent instructions like BIOS.
```

**AI Reference Answer:**
```
RAM (Random Access Memory) is volatile memory that temporarily stores data and programs currently in use. It loses all data when power is turned off. ROM (Read-Only Memory) is non-volatile memory that permanently stores critical system instructions like the BIOS/firmware. ROM retains data even without power and cannot be easily modified.
```

### 5. 💰 Economics
**Topic:** Inflation

**Question:**
```
What is inflation and what are its main types?
```

**Sample Student Answer:**
```
Inflation is when prices rise over time. Main types are demand-pull and cost-push inflation.
```

**AI Reference Answer:**
```
Inflation is the sustained increase in the general price level of goods and services in an economy over time, resulting in decreased purchasing power. The main types are: 1) Demand-Pull Inflation (when demand exceeds supply), 2) Cost-Push Inflation (when production costs increase), 3) Built-In Inflation (wage-price spiral), and 4) Hyperinflation (extremely rapid price increases exceeding 50% per month).
```
